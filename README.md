# Dance Website

A modern and responsive multi-page dance website built using Next.js and React. This project focuses on clean UI design, structured navigation, and basic user interaction features.

## Features
- Multi-page navigation (Home, About, Services, Contact)
- Clean and responsive user interface
- User login functionality
- Component-based architecture using React
- Fast performance with Next.js routing

## Tech Stack
- Frontend: React.js, Next.js
- Styling: CSS
- Tools: Git, GitHub

## Project Overview
This project was developed to practice modern frontend development using Next.js. It demonstrates how to build a structured website with multiple pages and reusable components while maintaining a smooth user experience.
